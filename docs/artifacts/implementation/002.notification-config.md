## 설계 대비 변경
- config_test.go의 notification env variables 테스트에 `require.NotNil` 검증을 추가하여 nil pointer dereference를 방지. 설계 명세에는 assert만 사용했으나 포인터 필드 접근 전 nil 체크가 필요.
- notification_test.go에 TestFormatMentions_Empty 테스트를 추가하여 빈 입력 엣지 케이스 커버.
